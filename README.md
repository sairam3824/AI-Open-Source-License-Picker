# AI Open Source License Picker

A rules-based web app that helps developers choose the right open source license for their project. Answer 5 questions, get your top 3 license matches with plain-English explanations, side-by-side comparison, and one-click copy.

No signup. No AI APIs. Just logic.

## Features

- 5-question wizard covering commercial use, copyleft, patent protection, and more
- Top 3 license recommendations with scoring and reasoning
- ELI5 (plain English) explanations for every license
- Side-by-side comparison table
- Full compatibility matrix across all 8 licenses
- One-click license text copy
- "Compare all licenses" modal

## Licenses Covered

MIT, Apache 2.0, GPL v3, LGPL v3, MPL 2.0, BSD 2-Clause, AGPL v3, The Unlicense

## Tech Stack

- [Next.js 16](https://nextjs.org) (App Router)
- [Tailwind CSS v4](https://tailwindcss.com)
- [Lucide React](https://lucide.dev)
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
